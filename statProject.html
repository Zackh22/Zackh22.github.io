Finding the Formula of Success for English Premier Teams

About the Project

This project relates to my Capstone Project because I was trying to calculate/predict which variables correlate with teams that make it to the playoffs. The two are also similar because we need to find a way to determine how to assign professors classes and class times. By determining the most impactful variables, like core requirements and room availability, we are able to start to piece together an algorithm to solve this problem.
