About the Project

This project is about creating and drawing lines in computer graphics.

Computer Graphics Shapes Curves Live Demo

This is a Chess Engine that I developed in the summer of 2021. It uses a minimax search with Alpha-Beta pruning to search through board states, and evaluates them with a comparator. It looks X turns forward and picks the move that produces the most preferable board state it can force its opponent into within that scope. It plays around a 700 level on the Chess Elo Rating Scale. I wrote this in Java using no external packages, entire from scratch.
